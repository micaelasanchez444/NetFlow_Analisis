# NetFlow – Análisis de Clientes y Predicción de Bajas

Proyecto de análisis de datos y machine learning aplicado a un conjunto de datos de una empresa proveedora de internet ficticia llamada NetFlow.

El objetivo del proyecto es analizar el comportamiento de los clientes, identificar patrones asociados a la permanencia y la baja del servicio, y construir un modelo predictivo orientado a la detección temprana de clientes con riesgo de abandono (churn).

## Contenido del proyecto
- Carga, exploración y limpieza de datos
- Análisis exploratorio y estadístico
- Visualización de datos (distribuciones, análisis geográfico y boxplots)
- Preprocesamiento de variables
- Modelado predictivo mediante Regresión Logística
- Evaluación del modelo y análisis de resultados
- Propuestas de aplicación en un contexto real
